### Basics of JavaScript
#### 1. Dynamically typed programming language
#### 2. Data type - primitive and non-primitive
##### a. primitive (immutable values) - number, string, boolean, null, undefined, symbol, BigInt
##### b. non-primitive (call by reference) - object, arrays, functions
#### 3. Variable declaration types - let, const, var
##### a. const - cannot be redeclared and cannot be reassigned
##### b. let - cannot be redeclared but can be reassigned
##### c. var - can be redeclared and can be reassigned
#### 4. String Methods - slice(), indexOf(), includes(), concat(), charCodeAt(), FromCharCode(), toUpperCase(), toLowerCase(), replace(), replaceAll(), repeat(), trim(), trimStart(), trimEnd()
