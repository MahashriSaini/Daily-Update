### Basics of JavaScript

1. Dynamically typed programming language

2. Data types
   - **Primitive (immutable values)**
     - Number
     - String
     - Boolean
     - Null
     - Undefined
     - Symbol
     - BigInt
   - **Non-primitive (call by reference)**
     - Object
     - Arrays
     - Functions

3. Variable declaration types
   - **const**
     - Cannot be redeclared
     - Cannot be reassigned
   - **let**
     - Cannot be redeclared
     - Can be reassigned
   - **var**
     - Can be redeclared
     - Can be reassigned

4. String methods
   - `slice()`
   - `indexOf()`
   - `includes()`
   - `concat()`
   - `charCodeAt()`
   - `fromCharCode()`
   - `toUpperCase()`
   - `toLowerCase()`
   - `replace()`
   - `replaceAll()`
   - `repeat()`
   - `trim()`
   - `trimStart()`
   - `trimEnd()`
